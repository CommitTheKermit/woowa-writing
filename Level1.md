## 글쓰기를 통해 발견한 내 모습

### 회고 쓰기로 찾아보기

우테코를 시작하고 나서 하루 하루가 귀중하다고 느껴졌습니다. 우테코에서 필요한 정보들을 배우고 코치, 리뷰어, 크루들과 소통을 하는 과정이 너무 즐거웠고 가치있다고 느껴졌습니다. 이 가치있는 하루를 단순히 끝내기보다 하루를 기록하고 돌아보면 좋겠다는 생각이 들어서 회고를 작성하기 시작했습니다.

집으로 돌아갈 때 회고를 작성하니 돌아가는 순간이 지루하지 않아져서 더 좋았습니다. 하루에 있는 일들을 기록하고 내가 어떻게 느꼈는지 작성해보니 배운 것들을 기억하기에도 좋았고, 내 감정을 자유롭게 얘기할 수 있는 공간이 있어서 속이 시원해질 때도 있었습니다.

회고를 통해 성찰을 이룰 때도 많아서 “조금 더 빨리 쓸 걸” 하고 생각했을 때도 많았습니다. 그러나, “우테코”라는 이벤트를 통해서 각성할 수 있어서 다행이라고 생각합니다.

### 블로그 쓰기로 찾아보기

레벨 1을 보내면서 3주차부터 블로그를 시작했습니다. 사실, 블로그를 시작하면서 순수하게 시작했느냐고 물어본다면 그건 아닌 것 같습니다. “나중에 취업에 도움될 것 같아서”, “이력서, 포트폴리오 이외에 나를 표현하고 싶어서”라는 이유로 시작했습니다.

처음에는 블로그에 글을 쓰고 싶어도 도저히 떠오르는 내용이 없어서 작성하고 싶은 글들이 없었는데, 미션을 수행하면서 생기는 의문들이 좋은 주제라고 생각했습니다. 블로그에 쓰려면 당연하게도 형식을 지키고 정리해서 제공해야 한다고 생각했습니다. 더해서, 잘못된 정보를 전달하지 않기 위해 혼자 공부할 때 보다 더 신경써서 정리했던 기억이 납니다.

이전에는 조사를 하더라도 가볍게만 조사하고 끝까지 진행하지 않을 때도 많았는데, 블로그 글을 쓰려고 하니 자연스레 끝까지 조사하게 될 때가 많아서 공부가 더 잘 되는 것 같은 기분이 듭니다.

막상 블로그를 시작하니 욕심이 나서 블로그를 어떻게 잘 작성할 수 있을지 고민이 생겼습니다. 원온원을 통해서 답을 얻었는데 “처음부터 너무 잘 작성하려고 하지말고 막 써라”, “다른 사람들에게 공유해서 피드백을 받아라” 라는 조언을 해주셨습니다. 특히, 제가 쓴 글 중에 하나를 콕 집어서 슬랙에 공유하라고 하셨습니다. 공유할 때는 무서웠는데 막상 한 번 해보니 다음은 쉬워서 신기했습니다.

지금은 미션이나 수업을 하다가도 블로그에 쓸만한 내용을 발견하면 블로그에 올리고 싶어서 더 열심히 공부하고 더 열심히 조사하게 되었습니다.

처음에는 순수한 의미로 시작한 건 아니지만, 블로그를 쓰면서 제 의도가 순수하게 변했습니다. 이제는, 다른 사람이 제 글을 보고 통찰을 얻었으면 좋겠고, 블로그를 통해 선한 영향력을 퍼뜨리고 싶습니다!


### 쓰레기를 잘 만드는 사람

회고에서 미션과 페어 프로그래밍에 대해 돌아보니 저는 “쓰레기를 잘 만드는 인간”이라는 생각이 들었습니다.

요구사항이나 문제를 해결하는 최소사항을 만족하는 결과물은 잘 만드는데, 그것 이상을 만들기가 어렵다는 생각이 듭니다.

쓰레기를 만들면서 “지금보다 더 나은 방법이 있을 것 같다”, “당장의 편의를 위해서 개발하는 것 같다”라는 생각이 들었고, 코드에 내 철학이나 컨셉을 담지 않고 관성적으로 개발하고 있다고 생각합니다. 결국에 저는 "쓰레기를 잘 만드는 사람"이 아니라 쓰레기를 통해 좋은 결과물을 만들고 싶은 사람인데 그게 어렵다고 생각이 들었습니다.

크루, 코치, 리뷰어들에게 조언을 구하였는데 “쓰레기를 잘 만드는 사람”도 실무에 필요하다고 해주셔서 응원이 되었습니다. 프로토타입이나, MVP를 잘 만드는 사람도 분명히 필요하니까요.

여러 의견을 듣고 제 생각을 정리해보니 레벨1 동안 열심히 한 것처럼, 지금처럼만 고민하고 배우면서 앞으로 남은 레벨들을 진행하면 발전할 수 있겠다는 자신이 생겼습니다.

### 성급한 도전을 하는 사람

안드로이드 크루들과 안드로이드 컨퍼런스에 참여했었습니다. 컨퍼런스 마지막에는 실습한 내용을 발표할 수 있는 기회가 있었는데, “발표를 하겠다”라는 생각만 가지고 준비를 하지 않고 발표를 하니 발표가 엉망이었습니다.

물론, 발표에 대한 반응이 나쁘진 않았다고 생각합니다. 오히려, “그럴 수 있다”라고 생각하시는 분이 많았을 것 같습니다.

그럼에도, 스스로를 돌아보니 너무 제 자신이 실망스러웠습니다. 실습에 급급하여 발표 내용에 대해서 준비를 하지 않았더니 발표 내용도 빈약하고 너무 많이 더듬어서 제가 말하고 싶은 내용을 매끄럽게 전달하지 못한 것 같습니다.

생각해보니, 저는 일을 벌리는 건 좋아하지만 그걸 잘 책임지지는 못하는 것 같습니다. 그거에 대한 준비나 마무리를 잘 수행하지는 못하는 것 같습니다. 조금 더 생각해보고 결정했어야 했는데 조금 성급했던 것 같습니다.
